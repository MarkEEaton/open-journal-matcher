    This tool compares the similarity of user-submitted abstracts with abstracts from the Directory of Open Access Journals, and provides a list of the 5 top matching journals. It is not meant to be forensic (it won’t tell you what journal an abstract came from), but rather it is meant to be serendipitous (maybe you’ll discover a new journal that you like!).

    While other journal matching projects exist, to my knowledge none of them are both fully open source, and cover all disciplines. All of the code for this project is openly licensed. The code powering this site is available on Github, as is the code for spaCy, the underlying matching algorithm (both use an MIT license), Moreover, the content is open too: the journals from the Directory of Open Access Journals are all openly licensed.
